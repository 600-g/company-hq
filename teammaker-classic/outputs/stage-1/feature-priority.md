## 스프린트 계획

### Sprint 1: 오피스 + 팀 만들기 (2-3주)
> "빈 사무실에 책상을 놓고 AI 팀을 만들 수 있다"

- A1: 오피스 캔버스 렌더링
- A2: 줌/패닝
- B1: 책상 드래그앤드롭
- B2: 팀 정보 입력
- B3: AI 팀 구성 제안
- B4: 팀 구성 확인/수정
- D1: API Key 입력

**데모 시나리오**: 빈 오피스 → 책상 배치 → "마케팅팀" 이름과 설명 입력 → AI가 에이전트 제안 → 확인 → 데스크에 팀 표시

### Sprint 2: 업무 지시 + 단일 팀 시각화 (2-3주)
> "팀에게 일을 시키고 진행 과정을 볼 수 있다"

- C1: 채팅 인터페이스
- C2: 팀 자동 매칭
- C3: 팀 활성 애니메이션
- A3: 데스크 상태 비주얼
- C5: 상세 진행 패널

**데모 시나리오**: 채팅에 "블로그 글 써줘" 입력 → 마케팅팀 데스크 빛남 → 클릭하면 진행 상황 표시 → 완료 시 결과물 표시

### Sprint 3: 팀 협업 + 폴리싱 (2-3주)
> "여러 팀이 협업하는 모습을 볼 수 있다"

- C4: 팀 간 협업 시각화
- D2: 온보딩
- D3: 작업 히스토리
- 전체 UX 폴리싱

**데모 시나리오**: "신제품 런칭 캠페인 만들어줘" → 기획팀 활성 → 기획안이 디자인팀으로 이동(애니메이션) → 디자인팀 활성 → 결과물이 마케팅팀으로 이동 → 최종 결과
