# Step 1.4: Screen Structure

## 화면 목록

| # | 화면 | 경로 | 설명 |
|---|------|------|------|
| S1 | API Key 입력 | `/setup` | 최초 접속 시 API Key 입력 |
| S2 | 메인 오피스 | `/office` | 가상 오피스 캔버스 (핵심 화면) |

## S2: 메인 오피스 화면 (핵심)

```
┌──────────────────────────────────────────────────────────┐
│ [≡] TeamMaker            [줌+] [줌-] [맞춤]    [⚙ 설정] │
├────────┬─────────────────────────────────────────────────┤
│        │                                                 │
│ 팔레트  │          가상 오피스 캔버스                       │
│        │                                                 │
│ ┌────┐ │    ┌─────────┐         ┌─────────┐             │
│ │ 📦 │ │    │ 마케팅팀  │         │ 개발팀   │             │
│ │빈   │ │    │ 🟢 작업중│         │ ⚪ 대기  │             │
│ │책상  │ │    │ 👤👤    │         │ 👤👤👤  │             │
│ └────┘ │    └─────────┘         └─────────┘             │
│        │                                                 │
│        │              ┌─────────┐                       │
│        │              │ 디자인팀  │                       │
│        │              │ ✅ 완료  │                       │
│        │              │ 👤       │                       │
│        │              └─────────┘                       │
│        │                                                 │
├────────┴─────────────────────────────────────────────────┤
│ 💬 무엇을 도와드릴까요?                          [전송 →] │
└──────────────────────────────────────────────────────────┘
```

**구성 요소**:

### 상단 바
- 좌: 햄버거 메뉴 + 앱 이름
- 중: 줌 컨트롤 (확대/축소/화면맞춤)
- 우: 설정 버튼

### 왼쪽 팔레트
- "빈 책상" 드래그 아이템
- 접기/펼치기 가능
- 나중에 팀 템플릿 추가 가능한 공간

### 중앙 캔버스 (PixiJS)
- 2D 탑뷰 오피스 그리드
- 배치된 팀 데스크들
- 각 데스크 표시 정보:
  - 팀 이름
  - 상태 표시 (🟢 작업중 / ⚪ 대기 / ✅ 완료)
  - 에이전트 아이콘 (인원 수 표현)
  - 작업 중일 때 글로우 애니메이션
- 팀 간 협업 시 이동 애니메이션 (파티클/아이콘)
- 마우스 드래그: 캔버스 패닝
- 스크롤: 줌인/줌아웃
- 데스크 클릭: 상세 패널 오픈

### 하단 채팅바
- 텍스트 입력 필드
- 전송 버튼
- 클릭/포커스 시 확장 가능 (채팅 히스토리 표시)

## 반응형 고려 (MVP 이후)

MVP는 **데스크탑 전용** (1280px 이상)으로 구현합니다.
- 캔버스 최소 너비: 800px
- 팔레트: 접기 가능 (60px ↔ 200px)
- 채팅: 하단 바 ↔ 우측 패널 토글
- 상세 패널: 우측 슬라이드인 (320px)
