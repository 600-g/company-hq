# Step 0.3: Solution Outline

## 제품명 (가칭)
**TeamMaker** - AI 에이전트 팀을 만들고 일을 시키는 가상 오피스

## 한 줄 설명
> Gather 스타일 가상 오피스에서 책상을 배치하고 설명을 적으면 AI가 팀을 만들어주고, 일을 시키면 어떤 팀이 처리 중인지 실시간으로 보여주는 서비스

## 핵심 기술 기반
- **LLM**: Anthropic Claude API (사용자가 API Key 입력)
- **에이전트 프레임워크**: 자체 구현 또는 오픈소스 활용 (Claude Tool Use 기반)

---

## MVP 핵심 기능 3가지

### 1. 가상 오피스 캔버스
**화면**: 2D 탑뷰 사무실 공간 (Gather 스타일)

- 빈 사무실에서 시작
- 그리드 기반 공간에 팀 데스크를 자유롭게 배치
- 각 데스크는 팀을 나타내며, 상태에 따라 시각적으로 변화
  - 대기 중: 조용한 상태
  - 작업 중: 활성화 애니메이션 (빛남, 캐릭터 움직임 등)
  - 완료: 결과물 아이콘 표시

**상호작용**:
- 캔버스 줌인/줌아웃
- 데스크 드래그 앤 드롭으로 위치 변경
- 데스크 클릭 시 팀 상세 패널 오픈

### 2. 팀 생성 플로우
**경험 흐름**:
1. 사이드바 또는 팔레트에서 "빈 책상"을 드래그
2. 오피스 공간 원하는 위치에 드롭
3. 팝업/패널이 열림:
   - 팀 이름 입력 (예: "마케팅팀")
   - 팀 설명 입력 (예: "SNS 콘텐츠를 기획하고 작성하는 팀")
4. AI가 설명을 이해하여 팀 구성을 제안:
   - 자동으로 적절한 에이전트 역할 제안
   - 예: "콘텐츠 기획 에이전트" + "카피라이팅 에이전트" + "이미지 프롬프트 에이전트"
5. 사용자가 구성을 확인/수정 후 완료

**팀 구조**:
- 유연한 구조: 단일 에이전트도, 복수 에이전트 묶음도 가능
- 각 에이전트는 역할(role), 능력(capabilities), 도구(tools)를 가짐
- AI가 자연어 설명으로부터 이를 자동 구성

### 3. 업무 지시 & 실시간 시각화
**업무 지시**:
- 오피스 하단 또는 사이드에 채팅 인터페이스
- 자연어로 업무 지시 (예: "다음 주 인스타그램 포스팅 3개 만들어줘")
- AI가 적절한 팀을 자동 매칭하거나, 특정 팀에게 직접 지시 가능

**실시간 시각화**:
- **팀 활성 상태**: 작업 중인 팀의 데스크가 시각적으로 활성화
- **팀 간 협업 흐름**: 팀A가 팀B에게 결과를 넘기는 과정을 화살표/이동 애니메이션으로 표현
  - 예: "기획팀"에서 "작성팀"으로 기획안이 이동하는 모습
- 클릭하면 상세 진행 상황 확인 가능

## 사용자 여정 (MVP)

```
[최초 방문]
빈 사무실이 보임 → "나만의 AI 회사를 만들어보세요!" 온보딩

[팀 만들기]
책상 드래그 → 배치 → "고객응대팀: 고객 문의에 친절하게 답변" 입력
→ AI가 "메일 응답 에이전트 + FAQ 검색 에이전트" 제안
→ 확인 → 데스크에 팀 생성 완료!

[일 시키기]
채팅: "어제 들어온 고객 문의 3건 답변 초안 만들어줘"
→ 고객응대팀 데스크 활성화 (빛남)
→ FAQ 검색 에이전트가 먼저 작동 → 메일 응답 에이전트로 결과 전달
→ 팀 간 데이터 이동 시각화
→ 완료 → 결과물 채팅에 표시 + 데스크에 완료 표시

[결과 확인]
데스크 클릭 → 작업 히스토리, 결과물 확인
```

## 차별화 포인트

| 기존 도구 | TeamMaker |
|-----------|-----------|

| 작업 과정 블랙박스 | 실시간 시각화로 투명한 과정 |
| 기능 중심 | 경험 중심 (게임 같은 재미) |

## 기술 아키텍처 (초안)

### 프론트엔드
- 캔버스 렌더링: HTML5 Canvas 또는 PixiJS/Phaser (2D 게임 엔진)
- 실시간 상태 업데이트: WebSocket
- UI 프레임워크: React / Next.js

### 백엔드
- 에이전트 오케스트레이션: 자체 구현 (Claude Tool Use 기반)
- 실시간 통신: WebSocket 서버
- 사용자/팀/작업 데이터: PostgreSQL
- API: REST + WebSocket

### AI 레이어
- LLM: Anthropic Claude API (사용자 API Key)
- 팀 자동 구성: Claude에게 팀 설명을 주고 에이전트 구성 제안 받기
- 업무 라우팅: 자연어 지시를 분석하여 적절한 팀에 자동 할당
- 에이전트 간 협업: 파이프라인 방식으로 결과 전달
