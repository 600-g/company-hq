# Step 0.1: Raw Idea

## 한 줄 요약
Gather 스타일 가상 오피스에서 AI 에이전트 팀을 시각적으로 만들고, 일을 맡기고, 처리 과정을 실시간으로 지켜보는 서비스

## 핵심 컨셉
- AI 에이전트를 "팀"이라는 익숙한 개념으로 포장
- 가상 오피스 비주얼로 비개발자도 직관적으로 이해 가능
- 게임 같은 경험으로 진입 장벽을 낮춤

## 타겟 사용자
- 비개발자
- AI에 대한 이해도가 낮은 사람들
- 목표: AI를 쉽게 이해하고 활용할 수 있게 돕기

## 핵심 흐름 2가지

### 1. 팀 생성 플로우
1. 책상(데스크)을 드래그하여 가상 오피스 공간에 배치
2. 팀 이름과 설명을 입력
3. AI가 설명을 이해하여 적절한 에이전트 팀으로 구성

### 2. 업무 시각화 플로우
1. 채팅으로 자연어 지시 (예: "이 기능 만들어줘")
2. 어떤 팀이 현재 처리 중인지 가상 오피스에서 실시간 확인

## 팀 구조
- 유연한 구조: 단일 에이전트도, 여러 에이전트 묶음도 가능
- 예: "디자인팀"에 UI 에이전트 + UX 에이전트를 넣을 수도 있고, 단일 "디자인 에이전트"로 구성할 수도 있음

## Gather 스타일 비주얼의 가치
- 팀 구조를 공간적으로 배치하면 직관적으로 이해하기 쉬움
- 재미있고 게임 같은 경험이 사용자를 끌어당김
- 여러 에이전트의 상태를 한눈에 파악 가능

## 미정 사항
- 플랫폼 (웹/데스크톱/모바일)
- 기술 스택
