# CPO 의사결정 로그
> CPO가 관리, 전원 참조
> 주요 의사결정과 그 근거를 기록

---

## 형식

```
### [날짜] 결정 제목
- 배경: 왜 이 결정이 필요했는가
- 선택지: A vs B vs C
- 결정: 선택한 안 + 근거
- 영향: 어떤 팀에 영향이 가는가
- 결과: (실행 후 기록)
```
