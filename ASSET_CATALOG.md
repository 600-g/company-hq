## 레퍼런스에서 배울 것

### 캐릭터 시스템 (LimeZu + Sunnyside)
- 4방향: front, back, left, right
- 액션: idle, walk, run, sit, phone, type, carry, jump
- 프레임: idle 2~4f, walk 6~8f, 특수 3~6f
- 크기: 16×32 (LimeZu) 또는 32×32 (Sunnyside)

### 건물/타일 시스템 (Sunnyside + pixel_tiles)
- 벽: 3~4종 색상 × 패턴 조합
- 지붕: 좌경사, 우경사, 꼭대기 타일 3종 세트
- 창문: 일반, 불켜진, 커튼 → 3 상태
- 문: 닫힌, 열린 → 2 상태

### 자연물 (Sunnyside + Sprout Lands)
- 나무: 계절별 × 크기별 (sm, md, lg)
- 꽃: 색상별 × 성장단계
- 물: 타일 자동연결 (autotile)

### UI (Sunnyside)
- 9-slice 박스, 체력바, 커서, 감정표현 아이콘

## 스타일 가이드 (두근컴퍼니 고유)

### 색상 팔레트
```
배경:  #1a1a2e (야간), #2888c8 (주간), #f5a040 (석양)
피부:  #fcd9a8, #e8c898, #d4b088
머리:  #333333, #5a3a2a, #884422, #aa6633
셔츠:  #3366aa, #44aa66, #cc4444, #8844aa, #dd8833
바지:  #2a2a3a, #3a3a5a, #5a4a3a
건물:  #3a4a5a, #5a6a7a, #7a8a9a
강조:  #f5c842 (금색), #50d070 (초록), #60a0e0 (하늘)
```

### 픽셀 규칙
- 기본 단위: 16×16 타일
- 캐릭터: 16×32 (1×2 타일)
- 아웃라인: 검정(#000) 아닌 어두운 채도색 사용
- 그림자: 본색 대비 20~30% 어둡게
- 하이라이트: 본색 대비 15~25% 밝게
- 디더링: 최소한으로 (클린 스타일)
