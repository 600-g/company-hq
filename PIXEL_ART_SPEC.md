## 캐릭터 움직임 규칙

### 걷기 (필수)
- 스프라이트시트 기반 프레임 애니메이션
- 최소 4프레임 (좌발-중립-우발-중립)
- 방향: 좌/우 (flip으로 처리)
- 속도: 불규칙 (0.3~0.8px/frame)

### 서있기
- idle 애니메이션 (미세한 움직임, 2프레임)

### 절대 금지
- 이미지가 그대로 x축 이동 (떠다니는 느낌)
- 스프라이트 없이 Graphics로 그린 사람
- AI 생성 이미지를 그대로 스프라이트로 사용

### 메인 건물 (두근컴퍼니)
- 타일맵으로 조합 (벽/창문/지붕/입구 타일)

### 주변 건물
- 다양한 높이/스타일 (상점, 아파트, 카페 등)
- LimeZu 타일셋으로 조합

| 계절 | 나무 | 소품 | 특수 효과 |
|------|------|------|----------|

| 시간 | 하늘색 | 조명 | 특수 |
|------|--------|------|------|

- 눈: 눈송이 파티클
- 흐림: 구름 많이

## 타일맵 작업 방식

### Phaser Tilemap 사용
```typescript
// 타일맵 로드
this.load.image("tiles", "/assets/tileset.png");
this.load.tilemapTiledJSON("map", "/assets/login_map.json");

// 생성
const map = this.make.tilemap({ key: "map" });
const tileset = map.addTilesetImage("modern_exteriors", "tiles");
map.createLayer("ground", tileset);
map.createLayer("buildings", tileset);
```

### 맵 에디터: Tiled (https://www.mapeditor.org/)
- 무료, 오픈소스
- LimeZu 타일셋 로드 → 맵 편집 → JSON 내보내기
- Phaser에서 바로 로드

## 구현 우선순위

1. **LimeZu 에셋팩 다운로드 + 타일셋 정리**
2. **Tiled로 로그인 맵 제작** (건물+거리+나무 배치)
3. **Phaser Tilemap으로 로드** (LoginScene 교체)
4. **캐릭터 걷기 애니메이션** (스프라이트시트)
5. **날씨/계절 효과** (파티클 + tint)
6. **사무실 씬도 점진 교체**
